👑 CONCIERGE BOT — OWNER GUIDE (SEGA)

Full control: schedule, reminders, tasks, TAs, broadcasts.
Owner-only powers are locked to your account.

━━━━━━━━━━━━━━━━━━━━

🎛 MAIN MENU
📅 My Schedule (/upcoming) — your next sessions.
📣 Remind (/remind) — pick a staff member → pick a session → send a reminder now.
📝 Assign Task (/task) — assign a custom task + deadline to any staff. (Owner-only)
📢 Broadcast (/broadcast) — one message to ALL registered staff. (Owner-only)
⚙️ Settings — opens the settings menu.

━━━━━━━━━━━━━━━━━━━━

⚙️ SETTINGS MENU
🔄 Reload (/reload) — re-sync schedule from Google Sheets; offers to notify affected staff.
📊 Sync Status (/sync_status) — last sync time + event count.
🎓 Assign TA (/assignta) — assign a TA to a cohort (drives its homework checks).
➕ Add TA (/addta) — register a new TA (name + Telegram ID).
🔗 Set Link (/setlink) — save a consult booking link for a cohort.
🌍 Timezone (/timezone) — change your display timezone.

━━━━━━━━━━━━━━━━━━━━

⌨️ EXTRA COMMANDS (type them)
/setgroup — map a cohort → its group chat ID (so links auto-post there).
/listgroups — list all cohort → group mappings.
/clearlinks — wipe all saved consult links.
/testlog — write a test row to the completions sheet (health check).

━━━━━━━━━━━━━━━━━━━━

🗓 RUNS AUTOMATICALLY
• Reminders — 1h before lectures; 10:00 AM for consults (recipient's zone).
• Completion checks — 2h after every session (Yes/No → reason).
• Task check-ins — pre-deadline heads-up + Yes/No at the deadline.
• Weekly Saturday 5 PM nudge — set next week's link, or Done weekly tasks.
• Homework checks — 3 days after each lecture/seminar, sent to the cohort's TA.
• Auto-skip — any check-in unanswered for 24h is logged as Skipped.
• Schedule sync — pulls Google Sheets every few hours.

━━━━━━━━━━━━━━━━━━━━

📊 LOGGED IN GOOGLE SHEETS
• Completions Log + Dashboard — session & weekly Yes/No/Skipped + per-staff rollups.
• HW Checks Log + TA HW Stats — homework results + each TA's completion rate.
• Tasks Log — every task you assign and its outcome (Assigned → Done / Not done / Skipped).

━━━━━━━━━━━━━━━━━━━━

💡 TIPS
• Assign Task and Broadcast are yours alone — no other account can use them.
• After editing the Google Sheet, tap 🔄 Reload to push it live and notify staff.
• Homework checks only fire for cohorts with a TA assigned — use 🎓 Assign TA first.
• Set a cohort's group with /setgroup so booking links post automatically.
